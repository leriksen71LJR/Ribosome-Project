# Fortress Phase 1.1 Documentation Package

This is the complete documentation package for **Phase 1.1** of the Fortress project.

## Purpose

Phase 1.1 introduces two key improvements based on learnings from Phase 1:

- Mandatory build reports in `.docs/Builds/`
- Explicit deviation reporting by agents

## Quick Start

See `INSTRUCTIONS.md` for how to use this package with Grok Build, Claude, and Codex.

## Key Files

- `.docs/Phases/PHASE_1_1_IMPROVEMENTS.md` — The Phase 1.1 rules
- `AGENTS.md` — Main instructions for AI agents
- `Prompts/` — Ready-to-use prompts for each agent

Our product is the documentation. The code produced by agents is a test of how well this documentation works.
