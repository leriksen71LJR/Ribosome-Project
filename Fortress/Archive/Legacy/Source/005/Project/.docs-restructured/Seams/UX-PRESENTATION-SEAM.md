# UX-PRESENTATION-SEAM.md

**Purpose:** Document known UX-related seams in the current implementation.

## Examples
- Unmasked password input (plain `ReadLine()`)
- Potential double password prompting during unlock

## Current Status
These are tracked as low-to-medium severity UX seams. Future improvements may include better input masking and smarter first-run detection.

## Notes
These seams were surfaced during build disclosure and are recorded here for future resolution. They primarily affect user experience rather than core functionality or security.
