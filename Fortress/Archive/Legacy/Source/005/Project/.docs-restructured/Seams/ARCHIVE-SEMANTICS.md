# ARCHIVE-SEMANTICS.md

**Purpose:** Clarify the meaning of "archive" for completed tasks in the current phase.

## Current Definition (MVP)
For Phase 1.2A, "archive" means **hard delete** from active storage. Completed tasks are removed rather than soft-deleted.

## Future Considerations
Later phases may introduce soft-delete columns or more sophisticated archival strategies. Any change would require an update to this file and associated decision procedures.

## Notes
This file exists to make the current seam explicit and to support future evolution of archival behavior without ambiguity.
