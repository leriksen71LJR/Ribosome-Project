# Builds

Agent-generated build reports go here.

**Naming:** `BUILD-REPORT-YYYY-MM-DD-XXX.md` — `XXX` is a 3-digit sequence (`001`, `002`, …) per day. Include **agent name** in the report header. Pair with `REASONING-YYYY-MM-DD-XXX-{Agent}.md` (same `XXX`; agent slug in reasoning filename — see `.docs/Phases/PHASE_1_1_IMPROVEMENTS.md`).

**Required sections:** See Phase 1.1 and `Evaluation/EvaluationCriteria.md`.

This folder is the **only** `.docs/` subdirectory agents may write to during a build (`AGENTS.md` Rule 5).
