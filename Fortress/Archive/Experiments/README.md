# Experiments Folder Structure

This folder contains all research experiments related to the Ribosome / Fortress documentation architecture work.

## Organization Principle

Each major experiment has its own folder. Inside each experiment folder:

- **Core/** — The primary guidance document(s) intended for the Steward.
- **Analysis/** — Supporting analysis, Build Agent feedback, research reports, and working documents.
- **Archive/** — Older versions and superseded files (kept for historical reference).
