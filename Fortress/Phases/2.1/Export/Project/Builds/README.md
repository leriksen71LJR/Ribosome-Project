# Builds — Phase 2.1

Agents **may write** combined build reports here during shootouts.

**Pattern:** `BUILD-REPORT-YYYY-MM-DD-XXX-{Agent}.md`

**Spec:** `.documents/process/BUILD-DISCLOSURE.md`

This is the only writable documentation location under `Project/` during a build (besides `.codingAgent/` proposals). Code goes to the external shootout project root.
