# Effort-vs-Impact.md

**Type:** Analytical Perspective (Strategy)

**Purpose:** Evaluate proposals and changes based on the ratio of effort required versus expected impact.

## Key Questions
- Which changes or documentation improvements will deliver the highest value for the amount of work required?
- How should we prioritize limited time and attention?

## Application Guidance
Use this perspective when deciding what documentation improvements or new strategies to develop first. Prioritize high-impact, lower-effort changes early.

## Notes
This is one perspective among several. It can be used independently or combined with others (e.g., Risk Perspective).
