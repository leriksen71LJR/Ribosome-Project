# Builds

Agent-generated build reports go here.

**Specification:** [BuildDisclosure.md](../../BuildDisclosure.md) — the single authority for naming, all required sections, and completion checklist.

**Naming:** `BUILD-REPORT-YYYY-MM-DD-XXX-{Agent}.md` — one file per build. Date, 3-digit sequence, and agent slug are all required in the filename.

This folder is the **only** `.docs/` subdirectory agents may write to during a build (`AGENTS.md` Rule 5).
